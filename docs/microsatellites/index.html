Geneious Microsatellites Plugin 1.0 Tutorial

This tutorial was developed by the Biomatters team

The Geneious Microsatellite Plugin provides a user friendly tool that imports ABI fragment analysis files and allows you to visualise traces, fit ladders, call peaks, predict bins, display alleles in tabular format and export your data for further analysis.

For background information on microsatellites, you can refer to this Wikipedia page.

This tutorial covers all the core features of the Geneious plugin.

If you want to open the tutorial in a different window, click the new window button .
