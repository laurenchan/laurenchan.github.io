5.3 ArcMap. Formatting Maps

[Note: this is very very basic. I will add detail]

Now that you have your point plotted correctly in geographic space, you can pretty-up your map. First, if you want to change your basemap, go ahead and do so using Add Basemap as you did before. If you do this, you may want to either turn off (uncheck) or delete (right-click > remove) other basemap layers in the "List by Drawing Order" tab of the Table of Contents. This will keep your Table of Contents from getting too cluttered.

Assuming you have a basemap you like, you can change the look of your map as follows:

Change the symbol - click on the symbol for the layer, or right-click on the name of the point layer and select properties. You can change the symbol under "Symbology"
Add labels - if you want to add the "name" from your csv file as a label again select "properties" for the layer. Click on the labels tab and check "label features the same way". Under Label String, select the column header of the names you want to display. You can also change font etc. Click OK when you're done.
Layout format - Click on View menu > Layout view. This gives you the "print" version. You can resize the portion of the map printed and zoom in and out using the menu buttons. This may take some trial and error. Remember the "undo" function.
Once you have the map cropped as you'd like, you can add scale bars, compass bearings, etc. under the "Insert" menu. There are tons of different options. Pick something neat and clean and put it somewhere on the map where 1) it will not obscure important data and 2) not clutter the image.
When you are happy with your graph, save as a pdf. Go to File > Export map... and export as a pdf. Make sure you also save your ArcMap file in case you want to make more changes down the line. The pdf that's generated can be further edited in Adobe Illustrator if you need, and can also be inserted into a Word doc and then cropped if/as necessary.
